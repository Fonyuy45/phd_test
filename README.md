# Path Follower Challenge

A C++ implementation of a path-following algorithm for a mobile robot. This project was developed as part of a technical evaluation.

## Project Structure

* `src/`: Source code and header files.
* `zone.txt`: Input data defining the operational zone.
* `CMakeLists.txt`: Build configuration.

## Dependencies

* **CMake** (Version 3.22 or higher)
* **Eigen3** (Linear algebra library)
* **C++ Compiler** (supporting C++14/17)

## Building the Project

To build the application, follow these standard CMake steps:

```bash
mkdir build
cd build
cmake ..
make
./prog_test
